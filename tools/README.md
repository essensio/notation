# tools

Заготовка под инструменты вокруг грамматики (пока пусто).

Планируемое:

- **валидатор грамматики** — проверка, что [`../grammar.ebnf`](../grammar.ebnf)
  корректен как EBNF (нет недостижимых/неопределённых нонтерминалов);
- **проверка примеров** — прогон файлов из [`../examples/`](../examples/) против
  грамматики, чтобы примеры не расходились с нормативным синтаксисом.

Инструменты не дублируют эталонный парсер из `essensio` — их задача держать сам
файл грамматики и примеры согласованными.
